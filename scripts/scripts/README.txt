Running Obstacle Expansion
	1. Make sure the maps are configured properly.  The resolution and the origin need to match what Rviz reads
	2. Run $ python ObstacleExpansion.py
	3. Obstacle expansion publishes two maps to /lowResMap and /expandedMap
	4. Make sure to then deploy a Navigation method to initiate the function

Running Lab4
	1. Map environment in Stage or on Turtlebot
	2. Make sure the map is properly configured and reorient robot using 2D Pose Estimate if needed
	4. Wait for the terminal to print "waiting" repeatedly
	5. Choose a goal using the 2D Nav Goal button
